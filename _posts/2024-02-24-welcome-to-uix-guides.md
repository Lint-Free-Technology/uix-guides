## About This Site

UIX Guides is built with Jekyll and uses the beautiful Yat theme. This site is designed to provide comprehensive documentation and guides for user interface experience.

## Features

This Jekyll site includes:

- **Beautiful Design**: Modern, responsive theme with dark mode support
- **Easy Navigation**: Clear structure for finding documentation
- **Rich Content**: Support for code highlighting, tables, diagrams, and more
- **SEO Optimized**: Built-in SEO tags and sitemap generation
